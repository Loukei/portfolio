Detailed Description

Creates a new file with Multipart upload.

  • 使用Martipart方法上傳檔案
  • 此方法建議在5mb以下檔案使用，與Simple upload不同的是可以自訂metadata
  • 父類為GDriveFileTask，為通用的File類操作
  • 使用GDriveFileTask::finished信號表示上傳完成(或失敗)
  • 使用GDriveFileMultipartCreate::getResource()或GDriveFileMultipartCreate::getResource() 接收上傳完成後的network reply資料
