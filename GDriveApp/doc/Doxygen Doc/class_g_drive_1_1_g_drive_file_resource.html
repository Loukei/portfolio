Detailed Description

The metadata for a file. \reentrant.

  • File儲存檔案的元資料
  • GDriveFileResource負責把原本由QNetworkReply的json字串轉成使用function的方式存取資料
  • 內部使用QJsonObject保存資料，因為每次回傳的內容根據api指定的fields不同，有些內容不會回傳
  • 編譯器有RVO的關係所以不實作拷貝&移動建構子
