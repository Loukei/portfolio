Detailed Description

Creates a new file with Simple upload.

  • GDriveFileSimpleCreate 是實現google Drive api的Files: create方法中的simple upload
  • GDriveFileSimpleCreate，是一個功能性的類別:藉由建構式的參數呼叫QOAuth2AuthorizationCodeFlow 完成上傳檔案的任務，在內部處理好網路請求
  • 不要在任務完成前刪除作為parent的QOAuth2AuthorizationCodeFlow物件，否則會導致失敗
  • 完成任務後，會發出GDriveFileTask::finished()信號
  • 使用GDriveFileSimpleCreate::getReplyString()取出回傳的內容
  • 在連接GDriveFileTask::finished()的槽函式中使用QObject::deleteLater()刪除物件
