Detailed Description

A base class to handle all the Google Drive Api File method.

  • 處理File類型的基底類別
  • 使用GDriveFileTask::finished信號表示操作完成或失敗
  • errorString()回傳錯誤的字串
  • isComplete()表示任務完成與否，初始值為false
  • isFailed()表示任務是否失敗，預設為false
  • getExpBackoffSleepTime()用於計算指數退避所需時間，用來在錯誤發生時計算重新發送請求所需時間
