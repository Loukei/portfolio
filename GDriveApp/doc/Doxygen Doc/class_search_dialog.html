Detailed Description

The SearchDialog class is use to search drive file and show search result on QTreeView.

  • 用來提供搜尋的介面
  • 按下search按鈕後會發送SearchDialog::query()給mainwindow
  • mainwindow接收後會處理搜尋請求，產生GDriveFileSearch物件
  • GDriveFileSearch物件收到搜尋完成的內容後可以透過connect SearchDialog::onFileSearch_finished()信號來接收搜尋完成的內容
  • 回傳的json格式字串透過QJsonModel處理，用QTreeView來顯示
